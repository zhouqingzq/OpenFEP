# Pre-M3 Readiness Checklist

- Attention bottleneck is integrated into the main loop and has a current-round on/off acceptance artifact.
- At least three minimal narrative worlds run deterministically in the current benchmark suite.
- At least two transfer directions pass the required gate in the current benchmark suite.
- Personality and narrative validation show statistically significant group differences with current-round artifacts.
- Long-run soak regression passes without action-collapse or termination regressions.
- Snapshot loading remains backward compatible when newer attention and self-model fields are missing.
- Runtime lifecycle evidence is revalidated in the current round.
- Runtime family coverage is revalidated in the current round.
- The final Pre-M3 readiness report says whether the system is ready for M3.
