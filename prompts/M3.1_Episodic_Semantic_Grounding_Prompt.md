# M3.1 Episodic Semantic Grounding Prompt

## Scope

Implement deterministic episode-level semantic grounding for open narrative text. The system must extract reusable semantic evidence, grounded motifs, and provenance from surface, paraphrase, and implicit cues before appraisal.

## Non-Goals

- No external embedding model or stochastic semantic parser.
- No hand-authored acceptance-only labels that bypass episode evidence.
- No breaking changes to existing `NarrativeCompiler` callers.

## Canonical Seeds

- `31`
- `131`

## Acceptance Gates

- `schema`: semantic grounding payload survives dictionary round-trip.
- `determinism`: the same episode grounds to the same motifs and evidence.
- `causality`: disabling semantic grounding degrades paraphrase or implicit interpretation.
- `ablation`: compiler output quality drops when grounding-derived motifs are removed.
- `stress`: negated, ambiguous, and conflicting text stays bounded and traceable.
- `regression`: legacy narrative compiler behavior remains available for explicit lexical cases.

## Audit Risks

- Fake grounding that only appears in provenance and not in appraisal.
- Hard-coded cue inflation disguised as semantic generalization.
- Missing provenance that prevents post-hoc reconstruction of semantic claims.

## Repair Loop

1. Run milestone tests and artifact generator.
2. Treat any failed gate as blocking.
3. Repair the smallest causal layer first: grounding, compiler, then trace/report.
4. Re-run milestone tests and acceptance artifacts until the report status is `PASS`.
