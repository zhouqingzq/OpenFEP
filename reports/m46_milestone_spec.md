# M4.6 Milestone Specification

## Title

`M4.6: Longitudinal Stability and Style Differentiation`

## Scope

- Run multiple seeds across multiple style profiles.
- Measure restart stability, profile divergence, and recovery-retention.
- Quantify stability, reproducibility, non-trivial style separation, and perturbation recovery with explicit statistical thresholds rather than qualitative labels.
- Produce longitudinal trace, divergence, recovery, stress, and acceptance artifacts.

## Acceptance Gates

- `schema`
- `determinism`
- `causality`
- `ablation`
- `stress`
- `regression`
- `artifact_freshness`
- `icc_stability_threshold`
- `cluster_reproducibility_threshold`
- `effect_size_separation`
- `recovery_similarity_threshold`
